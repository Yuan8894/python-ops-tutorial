1. **01_system_basics**: 获取系统信息、CPU、内存等状态

4. **04_log_analysis**: 使用正则与结构化模块分析服务器日志
5. **05_monitor_alerts**: 服务存活监控与告警通知
6. **06_automation_cli**: 打造专属的运维命令行工具
